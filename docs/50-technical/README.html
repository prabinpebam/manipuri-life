Technical Design

The engine, architecture, and - most importantly - the pipeline that turns the fact database into shippable game content.

TL;DR

The technical heart is a content data pipeline: the structured Historical Database is the upstream source, and gameplay data (crops, NPCs, items, events) is generated or validated against it. Engine, tools, save system and platform targets support that spine.
