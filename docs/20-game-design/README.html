Domain 20

Game Design

Systems derived from what actually filled people's days - not Stardew mechanics with a Manipuri skin.

TL;DR

The core loop and every subsystem come straight from the historical record: rice cultivation, fishing, weaving, markets, relationships, marriage, festivals, and mandatory Lallup service. The endgame rewards belonging and legacy - e.g. contributing to a monument or a village irrigation system - not wealth.
